# Favour Alivitsa — Portfolio

Personal portfolio website for **Favour Alivitsa**, a backend and full-stack developer based in Nairobi, Kenya, building transactional systems for emerging markets.

## Tech Stack

- **Vite** + **TypeScript** + **React**
- **Tailwind CSS** + **shadcn/ui**
- **Syne** & **DM Mono** fonts

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Sections

- **Hero** — Name, tagline, and quick links
- **About** — Bio and highlights
- **Projects** — E-Receipts Web App (M-PESA) & Computer Vision Object Detection
- **Skills** — Languages, backend, cloud, security, and applied AI
- **Resume** — Work experience, education, and certifications
- **Contact** — Contact form and social links
